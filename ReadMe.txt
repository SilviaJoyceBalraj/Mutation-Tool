Before you get started please make sure that th following are done:

1. Java needs to be installed on your machine.
2. JUnit Jar file needs to be downloaaded and installed in the current directory.
3. Set Environment variable for the Jar file.
4. Extract the Source code of Mutation tool from the Zip folder.
5. Place the input file named jacuda.java outside the current directory.
6. Place all the remaining files in the current directory.
7. In the ReadInputFile.java file, add the path of the jacuda.java file and the path of the new mutant jacuda.java file in the current directory.
8. Compile and Run ReadInputFile.java
